## React Icon specification

Icons are specified using svgs and can be used in other componets through the system. They include a screen-reader friendly title.

### Component usage
```js
<Icon id="creation" color="#000000" />
```

### Using props to customize the text

Modify the text to see it change live:
```js live
<Icon id="creation" color="#000000" />
```
